# Versionierung und Kommentare im Editor für interaktive Kommunikation

>[!NOTE]
>
> Die interaktive Kommunikationsfunktion ist im Rahmen des Early-Adopter-Programms verfügbar. Senden Sie von Ihrer Geschäftsadresse eine E-Mail an `aem-forms-ea@adobe.com`, um den Zugriff anzufordern.

Interaktive Kommunikation (IC) ermöglicht Unternehmen das Erstellen dynamischer, datengesteuerter Dokumente für die personalisierte Kundenkommunikation. Um eine bessere Zusammenarbeit, Governance und kontrollierte Veröffentlichungs-Workflows zu unterstützen, bietet der Editor für interaktive Kommunikation Versionierungs-, Prüf- und Kommentarfunktionen.

Mit diesen Funktionen können Autoren mehrere Iterationen einer IC verwalten, das Feedback von Prüfern erfassen, zu früheren Versionen zurückkehren und während des gesamten Inhaltslebenszyklus ein klares Audit-Protokoll führen.

## Versionierung für interaktive Kommunikation

Mit der Versionierung im Editor für interaktive Kommunikation können Autoren:

- Erstellen mehrerer Versionen eines ICs

- Inkrementelle Änderungen mit aussagekräftigen Beschriftungen und Kommentaren speichern

- Auf eine frühere Version zurücksetzen

- Unterschiede visuell vergleichen

- Verwalten Sie saubere Governance für Prüfungs-, Genehmigungs- und Versionszyklen.

Dadurch wird sichergestellt, dass Autorinnen und Autoren IC-Designs experimentieren, iterieren und verfeinern können, während der gesamte historische Kontext erhalten bleibt.

## Erstellen einer Version einer interaktiven Kommunikation

Verwenden Sie die Versionierung, wenn Sie vor Aktualisierungen den aktuellen Status einer IC beibehalten möchten.

So erstellen Sie eine neue Version:

1. Öffnen Sie AEM und navigieren Sie zu Forms → Forms und Dokumente.

1. Wählen Sie die interaktive Kommunikation aus, die Sie versionieren möchten.

1. Wählen Sie im linken Bedienfeld Versionen aus.

1. Wählen Sie im Menü Aktion (mit drei Punkten) die Option Als Version speichern.

1. Geben Sie eine Versionsbeschriftung und einen optionalen Kommentar ein, der den Zweck oder die Änderungen in dieser Version beschreibt.

1. Speichern Sie die Version.

Der Versionsliste wird ein neuer Versionseintrag hinzugefügt, der den aktuellen IC-Status erfasst.

## Aktualisieren einer Version

Jedes Mal, wenn Sie die interaktive Kommunikation ändern und speichern, können Sie eine neue Version erstellen, um den aktualisierten Status zu erfassen.

So fügen Sie nach der Bearbeitung eine neue Version hinzu:

1. Öffnen Sie die IDE im Editor und nehmen Sie Ihre Aktualisierungen vor.

1. Führen Sie die Schritte unter Erstellen einer Version einer interaktiven Kommunikation aus.

1. Fügen Sie einen Titel und einen Kommentar hinzu, der erklärt, was sich geändert hat.

Dies hilft Teams, den Fortschritt zu verfolgen, und sorgt für Transparenz im Lebenszyklus.

## Auf eine frühere Version zurücksetzen

Wenn eine IC zu einer früheren Konfiguration zurückkehren muss:

1. Öffnen Sie Forms und Dokumente und wählen Sie die IDE aus.

1. Navigieren Sie zum Bedienfeld Versionen .

1. Wählen Sie die frühere Version aus, die Sie wiederherstellen möchten.

1. Klicken Sie auf Auf diese Version zurücksetzen.

Der IC wird in der ausgewählten Version wiederhergestellt, sodass Autoren unerwünschte Änderungen rückgängig machen oder Fehler beheben können.

## Vergleichen von zwei Versionen

Autoren können zwei Versionen visuell vergleichen, um Änderungen zu verstehen, die über Aktualisierungen hinweg vorgenommen wurden. Dies ist für Benutzeroberflächen-, Layout- oder inhaltsbezogene Überprüfungen nützlich.

Zu vergleichen:

1. Navigieren Sie zu Versionen für die ausgewählte IC.

1. Wählen Sie eine beliebige Version aus der Liste aus.

1. Wählen Sie Mit aktueller Version vergleichen aus.

Eine Vorschau zeigt die Unterschiede zwischen den beiden Versionen nebeneinander an.

## Hinzufügen von Kommentaren zu einer interaktiven Kommunikation

Durch das Kommentieren können Reviewer und Autoren direkt im Editor für interaktive Kommunikation zusammenarbeiten.

Hinzufügen eines Kommentars:

1. Wählen Sie die interaktive Kommunikation in Forms und Dokumente aus.

1. Navigieren Sie zum Bedienfeld Kommentare .

1. Geben Sie Ihr Feedback oder Ihre Notizen ein und klicken Sie auf Kommentar hinzufügen.

Kommentare tragen zur Optimierung von Workflows zwischen Autoren, Prüfern und Stakeholdern bei.

>[!NOTE]
>
>Bei Verwendung von Kommentaren in interaktiven Kommunikationen wird der eigenständige Workflow „Überprüfung und Genehmigung“ für Formulare automatisch deaktiviert.

Versionierung und Kommentare im Editor für interaktive Kommunikation bieten:

- Strukturierte Content Governance

- Vereinfachte Zusammenarbeit

- Nachverfolgbarer Änderungsverlauf

- Einfaches Rollback und Vergleichsfunktionen

Diese Funktionen unterstützen kontrolliertes Authoring, Compliance und Iteration während des gesamten Lebenszyklus der Kundenkommunikation.
