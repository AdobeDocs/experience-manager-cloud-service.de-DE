# Wartungsversionshinweise {#maintenance-release-notes}

Im folgenden Abschnitt finden Sie die technischen Versionshinweise für die Wartungsversion von Experience Manager as a Cloud Service im Zusammenhang mit der Aktivierung von Funktionen in 2025.10.0.

## Version 22758 {#22758}

Im Folgenden finden Sie eine Zusammenfassung der fortlaufenden Verbesserungen für die Wartungsversion 22758, die am 1. Oktober 2025 veröffentlicht wurde. Die vorherige Wartungsversion war Version 22450.

Die Funktionsaktivierung von 2025.10.0 bietet den vollen Funktionsumfang für diese Wartungsversion. Weitere Informationen finden Sie in der [Experience Manager-Versions-Roadmap](https://experienceleague.adobe.com/de/docs/experience-manager-release-information/aem-release-updates/update-releases-roadmap).

### Verbesserungen {#enhancements-22758}

* ASSETS-56227: Umbenennung des Modifikators „adobe-countdown-timer“.
* CNTBF-493: Aktualisierung der Version des content-backflow-Pakets auf 2.0.28.
* CQ-4361110: Granite-Übersetzungen.
* CQ-4356992: Neueste AEM- und Granite-Übersetzungen.
* GRANITE-56026: Verbesserung der API-Status-Code-Antworten für Berechtigungen.
* GRANITE-61015: Paket `org.apache.commons.io.channels` wurde der öffentlichen exportierten Liste hinzugefügt.
* GRANITE-61167: Felix-Protokoll wurde auf die neueste OSGI-Spezifikation aktualisiert.
* GRANITE-61167: Aktualisierung einer Reihe von Apache Felix-Abhängigkeiten.
* GRANITE-61169: Verbesserung der Prüfung auf geschützte Zeichenfolgen.
* GRANITE-61622: Aktualisierung einer Reihe von Apache Sling-Abhängigkeiten.
* GRANITE-61663: `com.adobe.granite.repository.indexdefs-1.0.2` zum Schnellstart hinzugefügt.
* GRANITE-61811: `com.adobe.granite.repository-2.0.0` zum Schnellstart hinzugefügt.
* SITES-32014: Externe Ereignisse werden überwacht, um Service-Registrierungen zu aktualisieren.
* SITES-34277: Korrektur blockiert Fehler in Übersetzungs-Workflows für Seiten.
* SKYOPS-108706: Paket mit Versionsumschalter wurde auf die aktuelle Version (ETag-Caching) aktualisiert.
* SKYOPS-114210: Aktualisierung auf die neueste Version des Pakets „aem.pss.service“.
* SKYOPS-86745: Aktualisierung auf Sling ResourceResolver 1.12.2.
* SKYOPS-119811: dispatcher-publish 2.0.258 wurde veröffentlicht.

### Behobene Probleme {#fixed-issues-22758}

* GRANITE-61875: Trigger für „ungültige Ausdrucksauswertung“ wurden korrigiert. Erstellende konnten keine Inhaltsfragmente speichern und Assets konnten nicht heruntergeladen werden.
* SITES-22059: Behebung von JS-Fehler der Komponente „PDF-Viewer“. Nicht lokalisierte Zeichenfolge „File preview not available“ (Dateivorschau nicht verfügbar) in der Site „Kernkomponenten“ > „PDF-Viewer“.
* GRANITE-59704: Behebung von „htmllibmanager.debug“, wodurch der Bearbeitungsmodus fehlschlug.
* GRANITE-61042: Integration von FELIX-6796 (ServiceTracker NPE fix) in das AEM Felix-Web-Konsolen-Paket.
* GRANITE-61165: Workspace.copy() löst RepositoryException aus.
* GRANITE-60183: Aktualisierung von ui.commons auf 5.10.50.

### Bekannte Probleme {#known-issues-22758}

Keine.

### Eingestellte Funktionen und APIs {#deprecated-22758}

Veraltete und entfernte Funktionen und APIs in AEM as a Cloud Service werden im Dokument [Veraltete und entfernte Funktionen und APIs](/help/release-notes/deprecated-removed-features.md) beschrieben.

### Sicherheitskorrekturen {#security-22758}

Mit AEM as a Cloud Service sollen Sicherheit und Leistung Ihrer Plattform optimiert werden. In dieser Wartungsversion wurden 13 identifizierte Schwachstellen behoben, was für einen noch robusteren Systemschutz sorgt.
