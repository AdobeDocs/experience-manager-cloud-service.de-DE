      <ul>
        <li><a href="https://experienceleague.adobe.com/docs/experience-manager-cloud-service/content/forms/adaptive-forms-authoring/authoring-adaptive-forms-foundation-components/create-an-adaptive-form-on-forms-cs/creating-adaptive-form.html?lang=de">Erstellen eines adaptiven Formulars</a></li>
        <li><a href="https://experienceleague.adobe.com/docs/experience-manager-cloud-service/content/forms/adaptive-forms-authoring/authoring-adaptive-forms-foundation-components/create-an-adaptive-form-on-forms-cs/themes.html?lang=de">Gestalten eines adaptiven Formulars</a></li>

        <li><a href="https://experienceleague.adobe.com/docs/experience-manager-core-components/using/adaptive-forms/sample-themes-templates-form-data-models-core-components.html?lang=de">Verwenden von einsatzbereiten Designs und Vorlagen</a></li>
      </ul>
    </div>

<div class="feature-item">
      <h4>Headless Adaptive Forms</h4>
      <p>Erstellen und rendern Sie Formulare nativ auf einer Website, in Anwendungen oder in nicht visuellen Interaktionen:</p>
      <ul>
        <li><a href="https://experienceleague.adobe.com/docs/experience-manager-headless-adaptive-forms/using/overview.html?lang=de">Einführung in Headless Adaptive Forms</a></li>
        <li>Erstellen von Formularen in Ihrer bevorzugten Programmiersprache (z. B. React)</li>
        <li>Native Integration von Formularen in Desktop- und Mobile Apps, Websites und Chat-Anwendungen</li>
        <li>Wiederverwenden Ihrer proprietären UI-Komponenten mit Formularanwendungen</li>
        <li>Nutzung der Leistungsfähigkeit von Adobe Experience Manager Forms mit Frontend-Flexibilität</li>
      </ul>
    </div>
  </div>

<div class="feature-category">
    <div class="feature-item">
      <h4>Edge Delivery Services für Forms</h4>
      <p>Erstellen und liefern Sie leistungsstarke Formulare mit außergewöhnlichem Benutzererlebnis:</p>
      <ul>
        <li><a href="/help/edge/docs/forms/universal-editor/overview-universal-editor-for-edge-delivery-services-for-forms.md">WYSIWYG-Authoring mit dem universellen Editor</a> - leistungsstarke visuelle Benutzeroberfläche zum Erstellen von Formularen</li>
        <li><a href="/help/edge/docs/forms/create-forms.md">Dokumentenbasiertes Authoring</a> - Erstellen Sie Formulare mit bekannten Tools wie Microsoft Excel und Google Sheets</li>
        <li>Erweiterter Regeleditor zum Erstellen komplexer Formularlogik</li>
        <li>Erzielen nahezu perfekter Google Lighthouse-Werte durch optimiertes Laden von Formularen</li>
        <li>Schnellere Bereitstellung von Formularen mit minimaler Entwicklungszeit</li>
      </ul>
    </div>

## Early Adopter-Funktionen {#early-adopter-features}

<ul class="early-adopter-list">
    <li><strong>AEM Forms-KI-Assistent (Gen-KI)</strong> - Schnellere Erstellung von Formularen mit KI-gestützten Vorschlägen</li>
    <li><strong>AEM Forms Workfront Fusion Connector</strong> - Automatisieren von Workflows, die durch Formularübermittlungen ausgelöst werden</li>
    <li><strong>Konversativer Forms</strong> - Erstellen von Chat-ähnlichen Formularerlebnissen auf jeder AEM Sites-Seite</li>
    <li><strong>WYSIWYG-Authoring für Edge Delivery</strong> - Erstellen von Formularen mit dem universellen Editor für Edge Delivery Services</li>
    <li><strong>AEM Forms-zu-Marketo-Connector</strong> - Integrieren von Formularübermittlungen mit Marketo Engage</li>
  </ul>

<p>Eine vollständige Liste der Early-Access-Innovationen und eine detaillierte Dokumentation finden Sie auf der Seite <a href="https://experienceleague.adobe.com/en/docs/experience-manager-cloud-service/content/forms/forms-overview/early-access-ea-features">AEM Forms Early-Access-Programm</a>.</p>
