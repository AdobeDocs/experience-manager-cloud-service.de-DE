# Universeller Editor – Versionshinweise für 2026.03.12 {#release-notes}

Dies sind die Versionshinweise für die Version vom 12. März 2026 des universellen Editors.

>[!TIP]
>
>Informationen zum Testen von **zukünftigen** Funktionen des universellen Editors vor ihrer Veröffentlichung finden Sie in den [Vorschauversionshinweisen zum universellen Editor](/help/release-notes/universal-editor/preview.md).

>[!TIP]
>
>Auf [dieser Seite](/help/release-notes/release-notes-cloud/release-notes-current.md) finden Sie die aktuellen Versionshinweise für Adobe Experience Manager as a Cloud Service.

## Neue Funktionen {#what-is-new}

* Komponenten können jetzt über das [Kontextmenü“ von Containern ](/help/sites-cloud/authoring/universal-editor/authoring.md#context-menu) werden.

## Funktionen des Early-Adoption-Programms {#early-adopter}

Wenn Sie die unten aufgeführten Funktionen testen und Ihr Feedback geben möchten, senden Sie eine E-Mail an Ihren Adobe Customer Success Manager über die mit Ihrer Adobe ID verknüpfte E-Mail-Adresse.

* Für Inhaltsfragmente wurde eine flache Kopie implementiert.

## Andere Verbesserungen {#other-improvements}

* Es wurde ein Problem behoben, bei dem Unterelemente zur Dokumenterstellung nicht den n-ten untergeordneten Regeln entsprachen.
